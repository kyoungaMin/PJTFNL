### 3. 대시보드 차트 헤더 UX 개선

#### 변경 내용

차트 카드 헤더를 B2B SaaS 패턴에 맞게 3행 구조로 재편

**Before**
```
[매출 실적 추이 · 설명]        [월간 예측 →] [KPI 카드 박스]
```

**After**
```
매출 실적 추이                              월간 예측 →
단위: 백만원 · 실선=실적 · 점선=예측 밴드
다음달 P50 예측  ₩6,300M  ▲226.9% MoM
─────────────────────────────────────
차트
```

- KPI 박스 카드 제거 → 텍스트 기반 인라인 Summary KPI로 변경
- "월간 예측 →" 버튼을 제목 행 우측으로 이동
- `actual: row.actual ?? null` 처리 추가 — 실적 없는 구간에서 차트 급락 방지 (`connectNulls=false`와 함께 동작)
- 범례에 "예측만 표시 (실적 없음)" 항목 자동 표시 (null 행 존재 시)
